Windows Install Tutorial

CellVision has a one time package install using conda, a package manager bundled into miniconda. Package install without program install is by design - see section for advanced users below.

Download CellVision for Windows here, and extract it to anywhere that you like.

Miniconda

  • Install miniconda if you don’t already have Anaconda Navigator, using the .exe file here. This should also install Anaconda Navigator on your system.
  • Open Anaconda Navigator.

Adding Conda-Forge

  • On the Environment tab, click the Channels button to open the Channels Manager.
  • Click the Add button. Type conda-forge, and hit enter
    • ADD IMAGE
  • Hit Update Channels

Installing CellVision Packages

  • On the Environment tab, click the Import button on the bottom
  • Name the new environment CellVision, and click the folder icon.
  • Browse to the folder where you downloaded CellVision.
  • Important! - Change the dropdown in the lower right hand side to be Conda explicit specification file.(ADD IMAGE)
  • Click on the file requirements.txt and click Open.
  • Hit Import - CellVision’s packages should install without errors. This may take a while!

Running CellVision (Windows)

Config File

CellVision depends on a cvconfig.py file located in the directory where you downloaded CellVision. You will change the parameters in this configuration file to tell CellVision where to segment your images.

Important configuration settings

DIRECTORY_PATH - directory that contains your .tif bestFocus images (usually the bestFocus folder)
CHANNEL_PATH - path to your channels file (usually called channelNames.txt)
NUCLEAR_CHANNEL_NAME - name of nuclear stain (corresponding to channelNames.txt). Case sensitive.
OUTPUT_METHOD - (imagej_text_file, statistics, visual_image_output, visual_overlay_output, all)
BOOST - multiplier with which to boost the pixels of the nuclear stain before inference

Other settings are documented within the configuration file itself.

Modifying the configuration file

You can modify this file in any text editor of your choice. We will use Notepad++ for this tutorial.

Running CellVision

Now that the parameters have been set, we can easily run CellVision.

  • Open Anaconda Navigator.
  • On the Environment tab, click the CellVision environment and click the play button.
  • Under the dropdown, select open with Jupiter Environment (ADD IMAGE)
  • A browser window should open with a filesystem. Navigate to the CellVision folder that you downloaded.
  • Click on windows.ipynb. A new window will open. (IMAGE)
  • Click on the cell that starts from main import main.
  • Click the button on the top that says Run. CellVision should run and segment all images within the director you specified.

Advanced Users
